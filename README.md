# 🎯 Whack-a-Mole

A fun **Whack-a-Mole** game built with **React (Vite)**. Click the moles as fast as you can before the timer runs out!  

## 🛠 Features  
✔️ Randomly appearing moles  
✔️ Click to score points  
✔️ Game timer (30 seconds)  
✔️ Dark mode UI  
✔️ "Play Again" button
