# 🎯 Whack-a-Mole

A fun **Whack-a-Mole** game built with **React (Vite)**. Click the moles as fast as you can before the timer runs out!  

## 🛠 Features  
✔️ **Bigger & Animated Moles**  
✔️ **Smooth hover & click effects**  
✔️ **30-second timer**  
✔️ **Dark mode UI**  
✔️ **Play Again button**
